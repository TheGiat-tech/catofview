## Code Example

Here's a simple JavaScript function to calculate your cat's age in human years:

```javascript
function catAgeToHumanYears(catAge) {
  if (catAge === 1) {
    return 15;
  } else if (catAge === 2) {
    return 24;
  } else {
    return 24 + (catAge - 2) * 4;
  }
}

console.log(catAgeToHumanYears(5)); // 36 years
```
